# СRUD-приложение для создания постов
### Функционал приложения: 
- добавить пост;
- просмотр поста;
- редактирование поста; 
- удаление поста.
___
### Используемые технологии:
- язык PHP; 
- фреймворк Laravel 9; 
- СУБД MySQL;
- фреймворк Tailwind CSS; 
- Vue.js. 
 ___
### Для запуска проекта необходимо выполнить команды: 
1) php artisan serve
2) vite
